# Release v1.0.0-public-benchmarks

Public, server-side, reproducible parsing benchmarks for [NanoFDO](https://nanofdo.com).

## What is in this release

- `benchmark.py` — one-command reproducible benchmark client
- `RESULTS.md` — full methodology, raw runs, and summarized distributions
- `results.json` — machine-readable output generated by running `benchmark.py`

## Latest results (2026-07-04)

| Metric | NanoFDO | serde_json | speedup |

| p50    | 628 ns  | 2194 ns    | 3.49x   |
| p95    | 635 ns  | 2233 ns    | 3.51x   |
| p99    | 647 ns  | 2254 ns    | 3.48x   |
| mean   | 632 ns  | 2204 ns    | 3.49x   |

- **Server:** OVH KS-A — Intel Xeon E-2274G @ 4.0GHz — Roubaix, France
- **Payload:** 468 bytes JSON, 100,000 iterations measured server-side
- **Free tier:** 1,000,000 API requests/month

## How to reproduce

```bash
pip install requests
python benchmark.py your-email@example.com 5
```

Or register manually and call `POST https://api.nanofdo.com/api/v1/parse` with a Developer Edge key.

## About NanoFDO

NanoFDO is a high-performance JSON processing engine with an integrated L7 security layer.
Independently measured: 1,764 OWASP vectors embedded, 8/10 OWASP Top 10 2021 coverage, ~107 ns/scan, +0 ns overhead on legitimate requests.
